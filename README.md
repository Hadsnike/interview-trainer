# Дубль — тренажёр для подготовки к собеседованиям

Прогон собеседования по выбранной роли и уровню: интервьюер (GigaChat) задаёт
вопрос, вы отвечаете своими словами и сразу получаете разбор — оценку 0–10,
что зачлось, что провалилось, образцовый ответ и дополнительный вопрос.

## Стек
- **Backend** — Python + FastAPI, тонкий клиент к GigaChat (OAuth + chat/completions).
- **Frontend** — одностраничное приложение на ванильном JS, без сборки.
- **Без ключа** запускается в demo-режиме на заглушках — проект можно посмотреть сразу.

## Запуск

```bash
pip install -r requirements.txt
cp .env.example .env        # впишите свой GIGACHAT_AUTH_KEY (или оставьте пустым для demo)
cd backend
uvicorn app:app --reload
```

Откройте http://127.0.0.1:8000

`.env` подгружается автоматически при старте — переменные вручную экспортировать не нужно.

## Где взять ключ GigaChat
1. Зарегистрируйтесь на developers.sber.ru и создайте проект GigaChat API.
2. В проекте получите **Authorization key** (строка Basic, base64 от `client_id:client_secret`).
3. Вставьте её в `GIGACHAT_AUTH_KEY` в `.env`.
4. Для физлиц scope — `GIGACHAT_API_PERS`.

Сертификаты Минцифры часто не установлены в системе — для локальной разработки
оставьте `GIGACHAT_VERIFY_SSL=false`. В проде установите корневые сертификаты и
поставьте `true`.

## Публикация на GitHub

Ключ хранится в `.env`, который добавлен в `.gitignore` и в репозиторий не
попадёт. В git едет только `.env.example` — шаблон без секретов. Перед первым
коммитом убедитесь, что `.env` не отслеживается:

```bash
git status            # .env не должен быть в списке
```

Если он там есть — проверьте, что `.gitignore` лежит в корне и содержит строку `.env`.

## Структура
```
backend/
  app.py              FastAPI: /api/question, /api/evaluate, /api/health
  gigachat_client.py  OAuth-токен + вызовы chat/completions
  prompts.py          промпты генерации вопросов и оценки + парсинг JSON
frontend/
  index.html          весь интерфейс
```

## Идеи для развития
- Сохранение истории сессий и графика прогресса по темам.
- Голосовой ответ (распознавание речи) для тренировки устной формулировки.
- Режим «стресс-таймер» с ограничением на размышление.
- Экспорт разбора сессии в PDF.
